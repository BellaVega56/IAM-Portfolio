# IAM Engineering Portfolio

A 12-week hands-on Identity and Access Management engineering portfolio demonstrating proficiency with **Okta** and **Microsoft Entra ID**.

## About This Project

This repository documents a structured learning path from IAM fundamentals to production-ready engineering skills. Each week focuses on a specific theme, with parallel tracks for both major enterprise identity platforms.

**Themes covered:**
- Identity foundations and directory services
- Access control models (RBAC, ABAC, SoD)
- Authentication policies and Conditional Access
- Federation protocols (SAML, OIDC, OAuth 2.0)
- Provisioning and lifecycle management (SCIM, JML)
- Hybrid identity (AD integration)
- Automation (Workflows, Graph API, PowerShell)
- Privileged access management
- Identity governance and compliance
- Detection and security monitoring
- Infrastructure as Code (Terraform)
